template<typename... Components>
void ECSEngine::ECSEngine< Components >::Run ( )
inline

Runs the main game loop until all scenes are complete or window is closed.

Continuously runs the top scene in the scene stack. When a scene completes, it is popped from the stack. The loop continues until all scenes are complete or the window is closed. The engine handles the final display call on the window.
