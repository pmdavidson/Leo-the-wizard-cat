Detailed Description

template<typename... Components>
class ECSEngine::EntityManager< Components >

Manages entities and their components for the ECS engine.

and Validity

EntityIDs are valid until RemoveEntity() is called, after which they may be reused. GetComponent() returns a reference valid until the component or entity is removed. Component references are invalidated by RemoveComponent() or RemoveEntity().

Template Parameters
ComponentsThe component types that can be attached to entities.
