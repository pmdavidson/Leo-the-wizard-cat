Detailed Description

Packs textures into a single atlas and provides sprite/UV lookups.

and Validity

All textures and sprites are owned by SpriteManager for its lifetime. Sprite IDs returned from RegisterTexture() stay valid until the manager is destroyed. GetTexture() returns a reference valid for the manager's lifetime.
