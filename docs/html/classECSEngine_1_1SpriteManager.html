Detailed Description

Manages sprite resources and texture loading for the ECS engine.

and Validity

Textures are stored in an unordered_map and remain valid for the SpriteManager's lifetime. Sprites are stored in a vector and SpriteIDs remain valid for the SpriteManager's lifetime. GetSprite() returns a reference that remains valid until vector reallocation occurs. Storing long-lived references to sprites is unsafe if RegisterTexture() may be called later.
